# Installing Violyra skills for Windsurf

1. Clone this repository:
   ```bash
   git clone https://github.com/soasme/violyra.git
   ```

2. Copy or reference the skill rules in your project's `.windsurf/rules/` directory, or add a reference to the `skills/` directory in your Windsurf configuration.

## Available Skills

See [docs/installation.md](../docs/installation.md) for the full skill list and usage.
