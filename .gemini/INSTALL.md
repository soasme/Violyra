# Installing Violyra skills for Gemini CLI

Install as a Gemini CLI extension:

```bash
gemini extensions install https://github.com/soasme/violyra
```

Or clone and install locally:

```bash
git clone https://github.com/soasme/violyra.git
gemini extensions install ./violyra
```
