# Installing Violyra skills for Aider

1. Clone this repository:
   ```bash
   git clone https://github.com/soasme/violyra.git
   ```

2. Reference the skills in your `.aider.conf.yml`:
   ```yaml
   read:
     - path/to/violyra/skills/replicate/SKILL.md
     - path/to/violyra/skills/mv-compilation/SKILL.md
     # add other skills as needed
   ```

## Available Skills

See [docs/installation.md](../docs/installation.md) for the full skill list and usage.
