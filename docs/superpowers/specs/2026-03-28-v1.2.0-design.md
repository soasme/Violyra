# v1.2.0 Design Spec

Date: 2026-03-28

## Overview

v1.2.0 introduces two new model-calling skills (`using-replicate-model`, `using-falai-model`), removes the now-redundant `generating-seedance15-video` skill, and adds two new video prompt-writing skills (`writing-seedance20-prompt`, `writing-veo31-prompt`).

## Changes

### 1. New skill: `using-replicate-model`

Replaces and absorbs `skills/replicate/SKILL.md`. The existing `skills/replicate/` directory is deleted.

**`skills/using-replicate-model/SKILL.md`** covers:
- Authentication (`REPLICATE_API_TOKEN`)
- Workflow: choose model â†’ fetch schema â†’ create prediction â†’ poll â†’ return output
- Model discovery via search and collections APIs
- Running guidelines (official vs community models, concurrent predictions, URL expiry, webhooks)

### 2. New skill: `using-falai-model`

**`skills/using-falai-model/SKILL.md`** covers:
- Authentication (`FAL_KEY`)
- Five calling modes: run (sync), subscribe (SDK-managed polling), submit (manual async), streaming (SSE), real-time (WebSocket)
- Async queue lifecycle: submit â†’ poll status â†’ fetch response
- Webhooks (`?fal_webhook=` param, payload shape, retry behavior)
- Key differences from Replicate (auth header, model ID format, separate result endpoint, auto-retry)

**`skills/using-falai-model/references/`** â€” same one-file-per-model approach.

| File | Model ID | Use |

| `veo-3.1.md` | `fal-ai/veo3.1/text-to-video`, `fal-ai/veo3.1/reference-to-video` | Text-to-video and image-to-video |

| `chatterbox.md` | `fal-ai/chatterbox/text-to-speech` | TTS voiceover |

| `nano-banana-pro.md` | `fal-ai/nano-banana-pro` | Image generation and editing |

### 3. Remove `generating-seedance15-video`

Delete `skills/generating-seedance15-video/` entirely:
- `SKILL.md`
- `scripts/generate.js`
- `scripts/generate.__test__.js`

Coverage moves to `using-replicate-model` + `references/seedance-1.5-pro.md`. No project-specific wrapper script is needed â€” agents call the Replicate API directly using the reference as a guide.

---

### 4. Keep `writing-seedance15-prompt`

No changes to `skills/writing-seedance15-prompt/SKILL.md`.

---

### 5. New skill: `writing-seedance20-prompt`

**`skills/writing-seedance20-prompt/SKILL.md`**

Same structure as `writing-seedance15-prompt`. Covers:
- Image-to-video and text-to-video modes
- Prompt construction rules (motion first, sequential action, multi-character, camera language, intensity adverbs)
- Multi-shot scene support with explicit shot-switch syntax
- Seedance 2.0-specific differences from 1.5 (to be confirmed against model docs during implementation)

---

### 6. New skill: `writing-veo31-prompt`

**`skills/writing-veo31-prompt/SKILL.md`**

Covers:
- Text-to-video and reference-to-video (image-to-video) modes
- Veo 3.1-specific prompt guidance: cinematic realism, native audio generation cues
- Multi-shot scene support (Veo supports scene transitions natively)
- Prompt construction rules tailored to Veo's strengths (camera movement, lighting, physical realism)
- Failure cases and best practices
