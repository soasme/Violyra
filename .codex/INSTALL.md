# Installing Violyra skills for Codex

1. Clone this repository:
   ```bash
   git clone https://github.com/soasme/violyra.git
   ```

2. Symlink skills into your Codex skills directory:
   ```bash
   mkdir -p ~/.codex
   ln -s $(pwd)/skills ~/.codex/violyra
   ```

3. Skills will be available in your next Codex session.
